📥 Paso 1 — Guarda la llave en tu compu

💻 Si usas Windows:
  1. Abre PowerShell (menú Inicio → escribe "PowerShell").
  2. Copia y pega esto (crea la carpeta si no existe): 
    if (!(Test-Path "$HOME\.ssh")) { New-Item -ItemType Directory "$HOME\.ssh" }
  3. Copia el archivo jr_nube1_sp (el que Erik te dio) a esa carpeta. Puedes arrastrarlo con el explorador a: C:\Users\TU_USUARIO\.ssh\
  4. Protege la llave (copia y pega): 
    icacls "$HOME\.ssh\jr_nube1_sp" /inheritance:r /grant:r "$($env:USERNAME):(R)"
🍎 Si usas Mac o 🐧 Linux:
  1. Abre la Terminal.
  2. Crea la carpeta y mueve la llave: 
    mkdir -p ~/.ssh
mv ~/Downloads/jr_nube1_sp ~/.ssh/    # o donde la hayas guardado
chmod 600 ~/.ssh/jr_nube1_sp

🔌 Paso 2 — Conéctate

En la misma terminal (Windows PowerShell o Mac/Linux), escribe este comando:

ssh -i ~/.ssh/jr_nube1_sp jr@129.148.16.240

La primera vez te preguntará si confías en esa máquina (Are you sure you want to continue connecting?). Escribe yes y da Enter.

Después verás algo así:

jr@paxel-central:~$

¡Entraste! 🎉 Ya estás en tu espacio. Escribe ls para ver qué hay, o cat README.txt para leer el mensaje de Erik.

Para salir, escribe exit.
